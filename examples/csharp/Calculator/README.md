# Calculator

Exemplo de uma calculadora demonstrando traducao de metodos com parametros.

## Conceitos demonstrados

- Traducao de metodos com parametros (`Add,a:firstNumber,b:secondNumber` -> `Somar,a:primeiroNumero,b:segundoNumero`)
- Traducao de campos de classe (`OperationCount` -> `ContagemOperacoes`)
- Traducao de construtores
- Multiplas classes no mesmo projeto com anotacoes tradu

## Estrutura

- `Calculator.cs` - Classe principal com operacoes matematicas
- `Program.cs` - Ponto de entrada que demonstra o uso da calculadora
