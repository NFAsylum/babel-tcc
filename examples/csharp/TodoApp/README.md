# TodoApp

Exemplo mais completo de uma aplicacao de tarefas com multiplas classes.

## Conceitos demonstrados

- Projeto com multiplos ficheiros C#
- Traducao de classes, metodos, parametros e campos
- Uso de generics (`List<TodoItem>`)
- Uso de LINQ (`Where`, `FirstOrDefault`, `Count`)
- Traducao de strings literais em contexto

## Estrutura

- `TodoItem.cs` - Modelo de item de tarefa
- `TodoList.cs` - Gerenciador de lista de tarefas
- `Program.cs` - Ponto de entrada com demonstracao de uso
