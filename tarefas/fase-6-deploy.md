# Fase 6 - Polimento e Deploy

**Objetivo:** Preparar para lancamento no VS Code Marketplace.
**Pre-requisito:** Fases 1-5 concluidas.

---

## Tarefa 6.1 - Code cleanup e refatoracao final

**Escopo:**
- [ ] Remover codigo morto e TODOs pendentes
- [ ] Refatorar duplicacoes
- [ ] Revisar nomenclatura (consistencia PT/EN)
- [ ] Code review final de todo o projeto
- [ ] Garantir que todos os testes passam

**Entrega:** Codebase limpa e consistente.

---

## Tarefa 6.2 - CI/CD completo

**Escopo:**
- [ ] GitHub Actions:
  - Build automatico em push/PR
  - Testes automaticos
  - Gerar .vsix em releases
  - Publicar no Marketplace via tag
- [ ] Versionamento semantico automatico
- [ ] Release notes automaticas (baseadas em commits)

**Entrega:** Pipeline CI/CD completo e funcional.

---

## Tarefa 6.3 - Preparacao para Marketplace

**Escopo:**
- [ ] Criar conta de publisher no VS Code Marketplace
- [ ] Preparar icone da extensao (128x128 PNG)
- [ ] Preparar banner (1280x640)
- [ ] Escrever descricao atraente para o Marketplace
- [ ] Screenshots de qualidade (antes/depois, features)
- [ ] Categorias e tags relevantes

**Entrega:** Assets e metadata prontos para publicacao.

**Escopo:**
