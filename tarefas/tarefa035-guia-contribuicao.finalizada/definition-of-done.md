# Definition of Done - Tarefa 035

- [ ] `CONTRIBUTING.md` existe na raiz do repositorio
- [ ] Processo de contribuicao (fork -> PR) documentado passo a passo
- [ ] Padroes de codigo C# e TypeScript documentados com exemplos
- [ ] Guia de contribuicao de traducoes e claro e acessivel
- [ ] Template de bug report criado em `.github/ISSUE_TEMPLATE/bug_report.md`
- [ ] Template de feature request criado em `.github/ISSUE_TEMPLATE/feature_request.md`
- [ ] Template de PR criado em `.github/PULL_REQUEST_TEMPLATE.md`
- [ ] Codigo de conduta presente (pode ser referencia ao Contributor Covenant)
- [ ] Secao de code review com criterios claros
- [ ] Links para documentacao do desenvolvedor (docs/developer-guide/) funcionam
- [ ] Documento testado: um novo contribuidor consegue seguir os passos
- [ ] Nenhum placeholder ou TODO no arquivo
